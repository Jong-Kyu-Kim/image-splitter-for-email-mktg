# image-splitter-for-email-mktg
 
## Introduction 
이미지 파일을 불러와 Drag & Drop으로 링크 영역을 지정하고, 지정된 영역에 따라 분할된 이미지를 생성하여 레이아웃을 위한 Table 태그의 하위 요소에 삽입해 소스 코드를 생성합니다.   
이메일 마케팅을 위해 제작되는 HTML 파일을 위한 마크업 작업과 이미지 작업을 자동화하기 위해 구현 하였습니다.
